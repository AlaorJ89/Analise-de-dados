Projeto Análise de Vendas

Arquivos:
- base_vendas.csv: arquivo com os dados de vendas.
- main.py: script com análise e gráficos.
- Conclusoes.py: script que gera o relatório em PDF.
- Conclusoes.pdf: relatório gerado (opcional).

Como rodar:
1. Certifique-se de ter Python instalado.
2. Instale as bibliotecas necessárias:
   pip install pandas matplotlib fpdf
3. Rode o main.py para analisar os dados.
4. Rode o Conclusoes.py para gerar o PDF com as conclusões.
